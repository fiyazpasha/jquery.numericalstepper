Example usage of jquery.numericalstepper.js

The plugin provides a method called numericalstepper which is invoked on a container element. To download click here

$(".stepUpSlider").numericalstepper({
 stepUp : ".increase",
 stepDown : ".decrease",
 displayObject : ".stepUpCounter",
 start : 10,
 stepHandler: function(ns){
  $(".stepUpCounter").text(ns.value);
 },
 step: 1,
 min : 20,
 max : 100,
 eventHandler : "mousedown"
})
